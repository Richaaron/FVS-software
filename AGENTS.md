# AGENTS.md - FVS Software

## Project Overview
Folusho Victory Schools Result Management System with Backend

## Running the Application

### Backend
```bash
cd "C:\Users\PASTOR\Desktop\FVS Software"
npm install
npm start
```
Backend runs on http://localhost:3000

### Frontend
Open `index.html` in a web browser.

## API Endpoints
- POST /api/login - User authentication
- GET/POST /api/data - School data
- GET/POST /api/teachers - Teacher management
- GET/POST /api/students - Student management
- GET/POST /api/subjects - Subject management
- GET/POST /api/scores - Score management
